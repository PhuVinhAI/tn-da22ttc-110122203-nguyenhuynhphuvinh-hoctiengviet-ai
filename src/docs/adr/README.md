# Architecture Decision Records

ADR ghi lại các quyết định kiến trúc quan trọng của LinVNix theo format chuẩn [MADR](https://adr.github.io/madr/).

## Format

Mỗi ADR là một file `NNNN-slug.md` với các section:

- **Title** — tên quyết định, ngắn gọn
- **Status** — `Proposed` | `Accepted` | `Deprecated` | `Superseded by [link]`
- **Date** — YYYY-MM-DD
- **Context** — vấn đề cần giải quyết, ràng buộc, options đã xét
- **Decision** — quyết định được chọn
- **Consequences** — tác động tích cực và trade-off chấp nhận
